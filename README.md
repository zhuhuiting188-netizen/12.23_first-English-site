# 🚀 白金豪华版 AI 生活助理

这是一个专为高龄用户（80岁+）设计的全栈式智能桌面应用。它集成了笔记管理、待办清单、AI 深度聊天以及番茄专注计时器。

## ✨ 项目核心亮点

- **长辈适配设计**：采用超大字号、高对比度配色（白金幻彩风格），确保视力不佳的用户也能轻松阅读。
- **核心功能模块**：
  - **我的笔记**：支持分类管理，并具备 AI 自动总结、AI 辅助写作功能。
  - **待办事项**：极简的任务清单，支持标记完成与一键清理。
  - **AI 智能聊天**：支持用户直接与 LLM 交流，且能基于已有笔记内容进行精准问答（RAG 原型）。
  - **幻彩番茄钟**：全屏专注模式，带有人声/音效提醒功能。

## 🛠 技术架构

- **前端 (Frontend)**：原生 HTML5, CSS3 (包含动态渐变动画), JavaScript (ES6+)。
- **后端 (Backend)**：通过 Fetch 异步调用 OpenAI API 实现智能交互。
- **数据存储**：利用 `localStorage` 实现本地持久化，无需服务器即可保存用户数据。

## 📖 Git 实践记录

本项目开发过程中严格执行了以下 Git 标准操作：
1. **提交 (Commit)**：多次存盘记录，并成功同步至云端。
2. **回滚 (Reset)**：练习了撤销错误提交，保持代码版本整洁。
3. **分支 (Branch)**：创建独立分支开发新功能，确保主线稳定。
4. **合并 (Merge)**：完成功能开发后，将分支安全合回主干。
